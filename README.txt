The Snow Optics Toolbox: 
A set of MATLAB programs for viewing and analyzing images of snow 
developed for the study of the Bidirectional Reflectance Distribution Function (BRDF). 

This toolbox was developed by Andrew E. Slaughter at Montana State University.

Installation:
To install the software (1) download and unzip the file latest version from the downloads page, (2) open MATLAB, 
(3), change MATLAB's the current folder to the directory where the files were unziped, (4) type "optics" in the command line and (5) see the help.pdf file for a tutorial and complete documentation.
